# Title
A Unicorn's Wander

## Repository
<https://github.com/CharcoalFeather/Final-Project-Proposal.git>

## Description
For my final project I will create a platformer game with the objective of a player successfully navigating an area to reach a completion checkpoint. This project will showcase use of Pygame module features and implementation of artistic abilities in collaboration with code.

## Features
- Animated character
	- An animated character feature will be created by using the Pygame sprite module. 
- Key interaction
	- Keyboared interaction will be implemented using Pygame key modules. 
- Music
	- Music will be implemented using Pygame music module. 

## Challenges
- Main Menu implementation will need to be researched.
- Player Collision implementation will need to be researched. 
- Score counter implementation will need to be researched. 

## Outcomes
Ideal Outcome:
- The ideal outcome will be a short game level that allows a player to navigate a platform space to reach a finish line. 

Minimal Viable Outcome:
- The minimal viable outcome consists of a player being able to navigate a platform space.
